# Legal Base (Admin)

> Access only for platform admin users.

## Upload documents

1. Go to **/admin/knowledge**.
2. Set title, type, and description.
3. Optional: add a source URL.
4. Select the file and upload.

## Document types

- Legislation: laws, constitution, codes.
- Jurisprudence: court decisions.
- Doctrine: books and articles.
- Guidelines: internal guides.
- Template: document templates.
- Other: special content.

The type controls chunking for better retrieval.

## Status

- processing: indexing in progress.
- ready: available for queries.
- failed: indexing error.

## Delete documents

- Use **Delete** to remove an item from the corpus.
